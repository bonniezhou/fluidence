Fluidence

by Bonnie Zhou

Fluidence is a display typeface that captures the expressive flow of Chinese brush calligraphy in Latin letters. With its dynamic modulation, bold contrast, and abrupt directional shifts, it evokes the energy of an ink brush moving vividly to the whimsical temperament of its holder. A playful and modern typeface, Fluidence nods to the rich, timeless beauty of Chinese and East Asian culture. Fluidence is best suited for headlines, branding, posters, or any project that needs a touch of movement and charm.

What's a variable font?

A variable font is a single font file that contains multiple variations (width, weight, optical size, etc.) of a typeface. You can set the value of these variations using CSS. For example, the variable font 'Fluidence VF' contains all the weight variations of Fluidence, from Regular to Black. Here, weight is called an axis that can be set within an allowable range defined in the typeface. Using CSS, you can set Fluidence to have any weight within its defined range, from 400 (Regular) to 900 (Black).

In contrast, a static font represents only one specific style of the typeface. Before variable font technology, typefaces were produced as separate individual files, where each file would represent one specific width/weight/style combination. For example 'Fluidence Regular', 'Fluidence Bold', and 'Fluidence Black' would each be a separate file. This means that if you have a large font with many styles, you could end up with 20 or 30 different font files to represent a complete typeface.

Fluidence is available as individual static fonts ('Fluidence Regular', 'Fluidence Bold', and 'Fluidence Black') and also as a variable font with the complete weight axis ('Fluidence VF'). If you purchase the variable font, you also get the 3 static fonts as a package for free, and vice versa.
